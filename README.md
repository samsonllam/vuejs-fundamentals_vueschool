# vuejs-fundamentals_vueschool

It is the project from [Vue.js Fundamentals](https://vueschool.io/courses/vuejs-fundamentals)

#### Topics covered:

- Introduction to two-way data binding
- Template syntax and expressions
- Vue directives, loops and conditional rendering
- Vue Devtools
- Handling user Inputs
- Handling Events
- Vue.js Methods and Computed Properties
- Attribute Bindings and dynamic classes
- Components

The course is suitable for developers who do not yet know about Vue.js or are just getting started with Vue.
